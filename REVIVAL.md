# Grape Art Revival — Solana Graveyard Hackathon

**Track:** Art (Exchange Art, $5K)

## What Was Revived

Grape Art was a community-driven NFT art marketplace built on Solana by the Grape Protocol team. It featured artist verification, collection curation, and community governance of art listings.

## Key Changes
- Updated Solana dependencies to latest compatible versions
- Frontend application restored to functional state
